How to decompile apk files.

Download apk file using some tool/website (ex: http://apps.evozi.com/apk-downloader/)

run "extractEclipseProject.bat filename.apk ProjectFoder"

Depending on obfuscation complexity, tool will try to decompile conf and source codes.
You can go through scripts and take what you need.
You can also create apk file from extracted files after tweaking (like removing not-wanted download links etc)
